3 Data Hub

Watch these screen capture videos for assistance in navigating the Northeast Fisheries Science Center Marine Development GIS Data Hub.

3.1 Explore by Category

Explore datasets and maps by category.

3.2 View all Datasets

Use the gallery application to view all datasets, search datasets by name, open the dataset landing page, and view the dataset in ArcGIS Online Map Viewer.

3.3 Use the Interactive Mapper

Use the interactive mapping application to view NOAA NEFSC data alongside BOEM marine development data. Toggle data layers on and off, view the map legend, and click on the map for more information on map features.

3.4 Download Data

3.5 Filter Data

3.6 View Metadata
